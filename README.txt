Conversor de Monedas

Descripción
Este es un programa en Java que permite convertir entre diferentes 
monedas usando tasas de cambio en tiempo real. Utiliza la API de 
ExchangeRate para obtener las tasas de cambio actuales entre varias 
monedas.

Uso
-Al iniciar el programa, se muestra un menú de opciones para convertir 
entre distintas monedas.

-Selecciona una opción ingresando el número correspondiente y 
presiona Enter.

-Ingresa la cantidad a convertir.

-El programa mostrará el resultado de la conversión usando la 
tasa de cambio actual.

Ejemplo
Si eliges convertir de Dólar a Peso colombiano e ingresas 100, el 
programa te dará el resultado de la conversión según la tasa de 
cambio actual.
